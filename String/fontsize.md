#### fontsize 方法

  按照指定的尺寸来显示字符串。

##### 语法:

  ```javascript
  stringObject.fontsize( size );
  ```

##### 参数说明:

  size - 必需。从 1 至 7 的数字。

###### 示例:

  ```javascript
  var Str = "graybobo",
      s = Str.fontsize( 3 );
	  
  console.log( s );
  ```

###### 结果:

  ```javascript
  >>>
  <font size="3">graybobo</font>
  ```
