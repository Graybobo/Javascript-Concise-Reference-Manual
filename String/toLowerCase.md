#### toLowerCase 方法

  把字符串转换为小写。

##### 语法:

  ```javascript
  stringObject.toLowerCase();
  ```

##### 返回值:

  一个新的字符串，在其中 stringObject 的所有大写字符全部被转换为了小写字符。

###### 示例:

  ```javascript
  var Str = "Hello KILLHAPPY";
	  
  console.log( Str.toLowerCase() );
  ```

###### 结果:

  ```javascript
  >>>
  hello killhappy
  ```
