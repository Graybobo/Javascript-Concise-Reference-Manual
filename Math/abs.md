#### abs() 方法

  abs() -- 返回数的绝对值

##### 语法:

  ```javascript
  Math.abs( x );
  ```

##### 参数说明:

  x -- 一个数值
  
##### 返回值:

  x 的绝对值
   
###### 示例:

  ```javascript
  console.log( Math.abs( 11.31 ) );
  console.log( Math.abs( -11.31 ) );
  ```

###### 结果:

  ```javascript
  >>>
  11.31
  11.31
  ```
