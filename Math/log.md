#### log() 方法

  log() -- 返回数的自然对数

##### 语法:

  ```javascript
  Math.log( x );
  ```

##### 参数说明:

  x -- 任意数值或表达式
  
##### 返回值:

  x 的自然对数

##### 说明:

  x 必须大于 0
   
###### 示例:

  ```javascript
  console.log( Math.log( 5.3 ) );
  console.log( Math.log( 5 ) );
  console.log( Math.log( 0 ) );
  console.log( Math.log( -1 ) );
  ```

###### 结果:

  ```javascript
  >>>
  1.667706820558076
  1.6094379124341003
  -Infinity
  NaN
  ```
