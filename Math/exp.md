#### exp() 方法

  exp() -- 返回 e 的 x 次幂的值。

##### 语法:

  ```javascript
  Math.exp( x );
  ```

##### 参数说明:

  x -- 任意数值或表达式，被用作指数。
  
##### 返回值:

  返回 e 的 x 次幂。e 代表自然对数的底数，其值近似为 2.71828。
   
###### 示例:

  ```javascript
  console.log( Math.exp( 1 ) );
  console.log( Math.exp( -1 ) );
  console.log( Math.exp( 0 ) );
  console.log( Math.exp( 5 ) );
  console.log( Math.exp( 10 ) );
  ```

###### 结果:

  ```javascript
  >>>
  2.718281828459045
  0.36787944117144233
  1
  148.4131591025766
  22026.465794806718
  ```
