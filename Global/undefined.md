## Javascript undefined 属性

undefined -- 表示未定义的值

### undefined 语法:

  ```javascript
  undefined;
  ```

### undefined 说明:

undefined是全局属性，代表未定义的值。

###### 示例:

  ```javascript
  var u;
  console.log( u );
  ```

###### 结果:

  ```javascript
  >>> var u; console.log( u );
  undefined
  ```
