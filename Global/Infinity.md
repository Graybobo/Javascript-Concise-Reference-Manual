#### Javascript Infinity 属性

  Infinity -- 表示无穷大的数值属性

##### 语法:

  ```javascript
  Infinity;
  ```

##### 说明:

  Infinity 是一个全局属性，代表正无穷大的数值。Infinity 不是常量。( Number.POSITIVE_INFINITY 是常量 )

###### 示例:

  ```javascript
  console.log( Infinity );
  ```

###### 结果:

  ```javascript
  >>>
  Infinity
  ```
