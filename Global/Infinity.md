## Javascript Infinity 属性:

Infinity -- 表示无穷大的数值属性

### Infinity 属性语法:

  ```javascript
  Infinity;
  ```

### Infinity 说明:

Infinity是一个全局属性，代表正无穷大的数值。Infinity不是常量。( Number.POSITIVE_INFINITY是常量 )

###### 示例:

  ```javascript
  console.log( Infinity );
  ```

###### 结果:

  ```javascript
  >>> console.log( Infinity );
  Infinity
  ```
